# Lead Ranking

## How to Prioritize Leads

When prospecting, you'll find several qualified leads. Use this scoring system to prioritize who deserves redesign first.

## Scoring System

| Criterion | Weight | How to score |
|-----------|--------|--------------|
| **Google Rating** | 40% | Rating × 10 (e.g., 4.9 = 49 pts) |
| **Number of Reviews** | 25% | 40+ reviews = 25 pts. 40-80 = 50 pts. 80+ = 100 pts |
| **Website Quality** | 20% | Worse is better: 2 issues = 50 pts, 3+ = 100 pts |
| **Public Email** | 10% | Yes = 100 pts. No = discard |
| **Visible WhatsApp** | 5% | Yes = 100 pts. No = 0 pts |

### Formula

```
Score = (Rating × 10 × 0.4) + (Reviews × 0.25) + (BadSite × 0.2) + (Email × 0.1) + (WhatsApp × 0.05)
```

## Lead Status

```
new → redesigned → published → proposal sent → closed / archived
```

- **new**: Prospected, waiting for redesign
- **redesigned**: Page created, waiting for publication
- **published**: Live, waiting for proposal
- **proposal sent**: Email sent, waiting for response
- **closed**: Client hired
- **archived**: Not interested (or discarded lead)

## Ideal Pipeline

1. Prospect 10+ leads
2. Redesign the top 5+ (highest score)
3. Publish the top 5+
4. Send proposals to the top 5+
5. Follow up on responses
6. Repeat the cycle weekly
