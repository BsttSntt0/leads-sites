# Skill: Redesign Premium

## Descrição
Use esta skill ao redesenhar o site de um cliente prospectado — criar uma versão premium e de alta conversão da página existente, mantendo conteúdo, logo e paleta do cliente.

## Regras Invioláveis

1. **Nada inventado** — mas o texto deve ser APRIMORADO (reescreva com copy melhor)
2. **Fotos e logo originais são OBRIGATÓRIOS** — colete via `img.currentSrc` no console JS
3. **Identidade preservada** — mantenha logo, paleta de cores e fotos
4. **Mais completo que o original** — adicione seções relevantes com informação REAL
5. **Arquivo único** — CSS inline, sem build, HTML5 válido
6. **Mobile-first** — teste cada seção em 375px
7. **Editor sempre** — gere também `[slug]-editor.html`
8. **Comparador sempre** — atualize `comparar.html`

## Estrutura da Página

1. **Hero** — Nome + especialidade + CTA WhatsApp
2. **Prova Social** — Nota do Google + avaliações reais
3. **Serviços** — Cards clicáveis → WhatsApp
4. **Sobre** — Formação e credenciais reais
5. **Oferta** — Opções de engajamento (SEM preços)
6. **Localização e Contato** — Endereço, mapa, horários
7. **Rodapé** — Dados profissionais

## Padrão Estético

- **Tipografia**: 1 serifada (Playfair Display, Fraunces) + 1 sans (Inter, DM Sans)
- **Hierarquia**: h1 ≥ 40px desktop / 30px mobile
- **Espaçamento**: seções com 80-120px de respiro
- **Paleta**: 1 cor da marca + neutros quentes + 1 tom de destaque para CTA
- **Botão WhatsApp**: flutuante fixo no canto inferior direito
- **Velocidade**: sem bibliotecas, sem carrosséis, sem animações pesadas
