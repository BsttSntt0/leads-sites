# Ranqueamento de Leads

## Como priorizar leads

Ao prospectar, você encontrará vários leads qualificados. Use este sistema de pontuação para priorizar quem merece redesign primeiro.

## Sistema de Pontuação

| Critério | Peso | Como pontuar |
|----------|------|--------------|
| **Nota no Google** | 40% | Nota × 10 (ex.: 4,9 = 49 pts) |
| **Nº de avaliações** | 25% | Mínimo 40 = 25 pts. 40-80 = 50 pts. 80+ = 100 pts |
| **Qualidade do site** | 20% | Quanto pior, melhor: 2 problemas = 50 pts, 3+ = 100 pts |
| **E-mail público** | 10% | Sim = 100 pts. Não = descarta |
| **WhatsApp visível** | 5% | Sim = 100 pts. Não = 0 pts |

### Fórmula

```
Score = (Nota × 10 × 0,4) + (Avaliações × 0,25) + (SiteRuim × 0,2) + (Email × 0,1) + (WhatsApp × 0,05)
```

## Prioridade por Nicho

Alguns nichos historicamente convertem melhor:

| Nicho | Prioridade | Motivo |
|-------|------------|--------|
| Nutricionistas | 🔥 Alta | Alto ticket, site é vitrine essencial |
| Psicólogos | 🔥 Alta | Alta concorrência, site gera confiança |
| Dentistas | 🔥 Alta | Ticket alto, site é primeiro filtro |
| Advogados | ⚡ Média | Depende da área de atuação |
| Psiquiatras | ⚡ Média | Menos concorrência digital |
| Fisioterapeutas | ⚡ Média | Crescendo em presença digital |
| Personal trainers | 💡 Baixa | Ticket menor, Instagram resolve mais |

## Status do Lead

```
novo → redesenhado → publicado → proposta enviada → fechado / arquivado
```

- **novo**: Prospectado, aguardando redesign
- **redesenhado**: Página criada, aguardando publicação
- **publicado**: No ar, aguardando proposta
- **proposta enviada**: E-mail enviado, aguardando resposta
- **fechado**: Cliente contratou
- **arquivado**: Não teve interesse (ou lead descartado)

## Pipeline ideal

1. Prospecte 10+ leads
2. Redesenhe os 5+ melhores (maior score)
3. Publique os 5+
4. Envie propostas para os 5+
5. Acompanhe respostas
6. Repita o ciclo semanalmente
