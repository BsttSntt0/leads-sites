# Skill: Google Maps Prospecting

## Description
Use this skill when prospecting clients on Google Maps — find highly-rated businesses with bad websites, qualify leads, and build the lead list.

## The Ideal Lead Profile

- Rating **≥ 4.7** on Google with **≥ 40 reviews**
- **Has an active website that is BAD** — elimination requirement
- **Public email** on profile or website — elimination requirement
- Niches that work: professionals (nutritionists, psychologists, lawyers, psychiatrists, dentists, physiotherapists)

## The 3 Elimination Filters (apply IN THIS ORDER)

### 1. No own website → SKIP
The service being sold is REDESIGN: there needs to be a page to redesign.
**Does NOT count as own website**:
- Instagram/Facebook profile
- WhatsApp link
- Third-party directory page (Doctoralia, iFood, Linktree)
- Site offline or expired domain
- Record: "no own website" and continue

### 2. Good website → SKIP
If the page is modern, responsive, and well-structured, there's no pain to solve.
- Record: "website is already good" and continue

### 3. No public email → SKIP
The proposal is sent by email. Look for:
- Google Maps profile
- Website (contact pages, footer, privacy policy)
- If not found: record "no public email" and FIND ANOTHER lead

## How to Evaluate Website Quality

### BAD website (qualifies the lead) — 2 or more of these signs:

| Criterion | What to observe |
|-----------|-----------------|
| ❌ Not responsive | Breaks on mobile, tiny text, horizontal scroll |
| ❌ Outdated design | Decade-old layout, system fonts, stretched/pixelated images |
| ❌ No hierarchy | Everything same size, giant paragraphs, no visual path |
| ❌ No clear CTA | No visible contact button above the fold |
| ❌ Slow/heavy | Auto-sliders, old plugins, MB-sized images |
| ❌ Abandoned content | Old copyright, years-old news, broken links |
| ❌ Generic template | Empty sections, lorem ipsum, unrelated stock photos |

**Record the specific reason**: e.g., "not responsive + no CTA — phone only in footer"

## Execution Flow

1. Open Google Maps and search `[niche] in [city]`
2. Go through results in order, opening each listing
3. Note rating and review count; apply cutoffs (≥ 4.7 / ≥ 40)
4. Check if there's a website in the profile (Filter 1)
5. Open the site in a new tab and evaluate quality (Filter 2)
6. Hunt for email (profile + site) (Filter 3)
7. Collect qualified lead data: name, rating, reviews, phone/WhatsApp, email, website URL, specific reason, + 2-3 real review excerpts
8. Repeat until reaching the target or exhausting 25 establishments

## Best Practices

- Never re-evaluate a lead already in `leads.md`
- New rounds ADD to the same list, never duplicate
- Respect browser rhythm: open one tab per site, close after analysis
- If the city has few results, inform the user and suggest an alternative niche or nearby city
